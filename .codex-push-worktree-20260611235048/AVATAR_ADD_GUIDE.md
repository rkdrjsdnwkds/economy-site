# 아바타 추가 가이드

현재 Spark 요금제에서는 사이트 안에서 이미지 파일을 바로 업로드하는 방식 대신, **이미지를 인터넷에 올린 뒤 이미지 URL을 붙여넣는 방식**을 사용합니다.

## 아주 쉬운 사용 순서

1. 아바타 이미지를 인터넷에 올립니다.
2. 이미지 주소(URL)를 복사합니다.
3. 교사용 화면에서 `설정` 탭을 엽니다.
4. `아바타 등록·제작자 수익`에서 이름, 이미지 URL, 가격, 제작 학생을 입력합니다.
5. `아바타 등록`을 누릅니다.

등록하면 Firebase Realtime Database에 아바타 정보가 저장됩니다. 그래서 학교 컴퓨터와 집 컴퓨터가 같은 사이트를 열면 같은 아바타가 보입니다.

## 이미지 올리는 쉬운 방법

가장 쉬운 방법은 이미지 호스팅 사이트를 쓰는 것입니다.

1. 이미지 호스팅 사이트에 접속합니다. 예: Imgur 같은 이미지 업로드 사이트
2. 이미지를 업로드합니다.
3. 업로드된 이미지를 오른쪽 클릭합니다.
4. `이미지 주소 복사`를 누릅니다.
5. 복사한 주소가 `.png`, `.jpg`, `.jpeg`, `.webp`로 끝나거나, 브라우저에서 열었을 때 이미지만 보이면 좋습니다.
6. 그 주소를 사이트의 `이미지 URL` 칸에 붙여넣습니다.

## Google Drive를 쓸 때 주의

Google Drive 공유 링크는 보통 이미지 파일 자체 주소가 아니라 “미리보기 페이지 주소”라서 바로 안 보일 수 있습니다.

비전공자 기준으로는 Google Drive보다 **이미지 호스팅 사이트**를 추천합니다.

## 제작자 수익

- 기본 수익률은 판매가의 10%입니다.
- 설정 탭의 `아바타 제작자 수익률 %`에서 바꿀 수 있습니다.
- 학생이 아바타를 구매하면 구매 금액이 빠지고, 제작 학생에게 수익률만큼 자동 지급됩니다.
- 제작자가 자신의 아바타를 구매할 때는 제작자 수익을 따로 지급하지 않습니다.

## 파일로 여러 개 추가하기

많은 아바타를 한꺼번에 관리하려면 `catalog.js`의 `avatarItems`에 추가할 수 있습니다.

```js
my_avatar: {
  id: "my_avatar",
  name: "새 아바타",
  type: "avatar",
  icon: "A",
  price: 300,
  rarity: "일반",
  src: "https://example.com/my_avatar.png",
  creatorId: "학생ID"
}
```

학생ID를 모르거나 나중에 바꾸려면 교사용 설정 화면의 제작자 연결 표에서 선택하면 됩니다.
